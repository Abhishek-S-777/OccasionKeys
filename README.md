# OccasionKeys
OccasionKeys is an android app built using Java and XML with Google Firebase as back-end, to provide users all the necessary items required for an event such as  sound systems, tents, utensils etc.

### Key Features: ###

#### User Features ####
* Browse through various *products, add/remove products to cart*.
* Search for products using the **search bar**
* Update profile such as their **display avatar and other details**.

#### Admin Features ####
* **Add products** by selecting the appropriate categories.
* Manage products, such as **update and delete products.**
* View the order details and confirm order *shipped* where users will receive an SMS for confirmation.
